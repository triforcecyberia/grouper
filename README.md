# Grouper

Программа группирует строки CSV-файла по совпадающим значениям в одинаковых колонках.

## Требования

- Java 11+
- Maven 3+

## Сборка

```bash
mvn package
```

Собранный jar появится в папке `target/grouper.jar`

## Запуск

```bash
java -Xmx1G -jar target/grouper.jar /полный/путь/до/файла.txt
```

### Пример

```bash
java -Xmx1G -jar target/grouper.jar /home/user/lng-4.txt
```

## Входной файл

Файл можно скачать здесь:
https://github.com/PeacockTeam/new-job/releases/download/v1.0/lng-4.txt.gz

Распаковать архив и передать путь до `.txt` файла как аргумент.

## Результат

После запуска в текущей папке появится файл `result.txt` со сгруппированными строками.

Формат вывода:
```
Количество групп с более чем одним элементом: 1910

Группа 1
строка1
строка2
...

Группа 2
строка1
...
```

## Производительность

Протестировано на файле из ~1 000 000 строк:
- Уникальных строк: 998 578
- Групп с более чем одним элементом: **1910**
- Время выполнения: **~10 секунд**
- Память: в рамках `-Xmx1G`

## Алгоритм

Используется структура данных **Union-Find** (Disjoint Set Union) с оптимизациями:
- Path halving для ускорения поиска корня
- Union by rank для балансировки дерева
- Ручной парсинг строк вместо `String.split()` для минимизации аллокаций
