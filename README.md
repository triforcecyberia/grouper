## Grouper

Программа для группировки строк.

## Сборка

mvn package

Собранный jar появится в папке `target/grouper.jar`

## Запуск

java -Xmx1G -jar target/grouper.jar /полный/путь/до/файла.txt

## Пример

java -Xmx1G -jar target/grouper.jar /home/user/lng-4.txt

## Результат

После запуска в текущей папке появится файл `result.txt` со сгруппированными строками.

## Производительность

Протестировано на файле из ~1 000 000 строк:
- Уникальных строк: 998 578
- Групп с более чем одним элементом: 1910
- Время выполнения: ~10 секунд
- Память: в рамках `-Xmx1G`
